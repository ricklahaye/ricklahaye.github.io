[2] Automated Forensics for popular Cloud Storage clients - University of Amsterdam [2018]

A Proof of Concept of automated forensics on popular Cloud Storage clients. The implementation used fuzzy hashing to detect clients on filesystems, and then uses multiple tooling including Python to decode forensic artifacts like (deleted) file and folders, records, and personal account details. It supported OneDrive, Google Drive, iCloud Drive, and Dropbox.

Tools/Techniques that were used are as following:
- Context Triggered Piecewise Hashes (CTPH)/Fuzzy Hashing
- SQLite3
- Python-Registry
- DPAPI and DPAPI-NG: Decryption Toolkit

[3] Machine Learning-based Intrusion Detection System for Industrial Control Systems - University of Amsterdam/Deloitte [2017]

A Proof of Concept of an Intrusion Detection System (IDS) that is able to detect anomalies in a Industrial Control System (Water Thermostatic Closed Loop Environment) by using Machine Learning.

Machine Learning algorithms used:
- Novelty and Outlier Detection (One-Class Support Vector Machine)
- Neural Networks (Long Short Term Memory)

[4] NTP as a Covert Channel - University of Amsterdam [2017]

A Proof of Concept of a covert channel using the Network Time Protocol (NTP). This implementation was compliant with the NTP RFC resulting in a lower chance of detection, and had significant improvement in performance in comparison with other available solutions.

Benchmarks (1 Gbit link):
- No tunnel: 940 Mbps
- NTP Tunnel: 920 Mbps unencrypted
- NTP Tunnel: 360 Mbps AES encrypted
- Iodine: 100 Mbps
- Ptunnel: 7 Mbps
